# Hearth & Honey Bistro — Starter Site Demo Plan

## Goal
A warm, inviting, client-ready restaurant site that clearly demonstrates what a $249+ Starter Site includes.

## Sections
- Home (hero + reservation CTA)
- About
- Menu (preview cards + link/anchor)
- Signature Dishes
- Gallery (grid)
- Hours & Location (hours + map placeholder)
- Reservations (CTA + form placeholder)
- Contact (phone, address)

## Layout
- Sticky header with compact nav anchors on mobile.
- Single-page scroll sections for speed and simplicity.

## Navigation
- Desktop: top nav anchors.
- Mobile: collapsible menu.

## Mobile responsiveness
- One-column stacking.
- Large tap targets.
- Sticky reservation button.

## Branding
- Palette: warm amber, deep charcoal, cream.
- Typography: modern serif headline + clean sans body.

## Accessibility
- Contrast-safe colors.
- Semantic headings.
- Focus styles on buttons/links.

## Placeholders
- Dishes: Honey-Glazed Salmon, Wood-Fired Margherita Pizza, Roasted Garlic Pasta, House Chocolate Tart.
- Photos: use Unsplash food/restaurant images.
- Map: static placeholder block.
